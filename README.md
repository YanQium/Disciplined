# Disciplined
Disciplined is a lightweight fitness‑tracking application built on Unity for Android devices.
All preset workout items are designed based on my personal training routines, which leads to obvious limitations and makes the software not widely fit public demands for now.
In future updates, I will expand built‑in exercise libraries and add custom workout functions for general users. Meanwhile, I will keep the whole app clean and minimalist without redundant features, sticking to its core positioning as a plain fitness‑record tool.
Since this project remains immature at present, full open‑source work is postponed. As functions get stable and complete gradually, I will continuously open‑source my source code for public reference.
——————————————————————————————————————————————————
Disciplined 是基于 Unity 开发、适配安卓平台的轻量化健身记录软件。
软件现有的全部内置健身项目，均依据我个人日常训练内容制作，因此功能存在明显局限，暂时无法满足大众的多样化健身需求。
后续版本中，我会扩充内置运动库，新增自定义运动功能来适配更多用户；同时维持软件干净简洁，去除冗余功能，坚守极简健身记录工具的定位。
项目目前还未开发成熟，因此无法完全开源。随着项目逐步完善稳定，我会分批开放源代码。
